# UI 界面设计原型

> Electron 桌面应用的用户界面设计规范和原型

---

## 🎨 设计原则

### 1. 简洁直观
- 运营人员零技术背景也能轻松使用
- 每个页面单一职责，避免复杂操作

### 2. 实时反馈
- 所有操作立即给予视觉反馈
- 进度条、加载状态、成功/失败提示

### 3. 容错友好
- 清晰的错误提示和恢复建议
- 重要操作二次确认

### 4. 现代化
- 使用流行的设计语言（类似 Notion、Figma）
- 响应式布局，适配不同窗口大小

#### 交互流程

1. **第1步**: 填写基本信息
   - 活动名称
   - 选择区块链（单选）
   - 输入代币地址（自动验证并显示代币信息）

2. **第2步**: 上传地址列表
   - 支持拖拽上传 CSV
   - 实时验证格式
   - 显示总地址数和总金额

3. **第3步**: 预览确认
   - 显示前50个地址预览
   - 显示预估Gas费用
   - 二次确认

4. **第4步**: 创建成功并准备充值 ⭐ 新
   - 显示活动ID和钱包地址
   - 显示需要充值的金额
   - 私钥导出选项
   - 余额监控面板
